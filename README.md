# FX Live - Real-Time Currency Converter

A modern currency converter with live exchange rates. Built with vanilla HTML, CSS, and JavaScript.

**[Live Demo](https://rezacr588.github.io/currency-convertor/)**

## Features

- Real-time conversion between 30+ currencies
- Live rates dashboard with auto-refresh (60s)
- Searchable currency dropdowns
- Historical rate comparisons
- Dark mode, responsive design
- No API key required

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (vanilla)
- **API**: [Frankfurter API](https://www.frankfurter.app/) (European Central Bank data)
- **Hosting**: GitHub Pages

## Quick Start

```bash
# Clone and open locally
git clone https://github.com/rezacr588/currency-convertor.git
open index.html
```

Or deploy to GitHub Pages:
1. Fork this repo
2. Go to **Settings** > **Pages**
3. Select branch and save
